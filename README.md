- Motivo da nossa criação:

  Inicialmente pensamos em um projeto que tenha como foco principal ajudar pessoas com deficiência ou algo em que use programação de forma obrigatoria no nosso projeto, com isso coincidentemente o professor paulão tinha um colega que trabalha como cozinheiro e tem deficiencia visual e perguntou se tinha como ele criar ou se já tinha algum projeto sobre um termômetro que consiga ajudar ele com sua dificuldade já que os termômetros atuais so mostram a temperatura em uma tela embutida no proprio termômetro.

- Quais ferramentas usamos para a organização e a utilização prática e teórica do projeto:

  Nos decidimos usar o tinkercad como um site de teste para o circuito do termômetro, fizemos também um documento para apresentarmos de forma formal o projeto ao professor da area além disso usamos o codepen para fazer uma mini apresentação de forma simples e prática para apresentamos aqui no curso.
